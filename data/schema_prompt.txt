表名: 大桥局人员信息表
字段:
- ID (int)
- 档案编号 (string): 类似5eff5d2f-ad14-3336aed-9e93-f0ac281ab1e2的字符串，用档案编号查询就是用大桥局人员信息表的档案编号。人数就是大桥局人员信息表的档案编号个数
- 排序 (string)
- 所属部门 (string)
- 所属项目 (string)
- 姓名 (string)
- 岗位 (string): "单元长"可能出现在此字段（正常情况），也可能异常出现在【职务】字段。当 岗位 = '单元长' OR 职务 = '单元长' 时，判定为单元长
- 职务 (string): "单元长"可能出现在此字段（异常情况），也可能正常出现在【岗位】字段。当 岗位 = '单元长' OR 职务 = '单元长' 时，判定为单元长
- 手机号 (string)
- 状态 (string)
- 正式员工 (int)
- 安全责任单元长 (int)
- 用户名 (string)
- 选中 (int)
- 履职经历 (string)
- 所属稽查队 (string)
- FGC_Creator (string)
- FGC_CreateDate (string)
- FGC_LastModifier (string)
- FGC_LastModifyDate (date)
- FGC_Rowversion (time)
- FGC_UpdateHelp (string)

表名: 安全责任单元履职清单主表
字段:
- ID (int): 对应每日管控计划_子表.工序id字符串
- 名称 (string): 就是对应的工序的名称，工序有顺序
- 级别 (int): 就是顺序
- 上级ID (int): 对应安全责任单元履职清单主表.ID，就是指当前工序的上一级工序。举个例子：安全责任单元履职清单主表中ID为15，名称为路基开挖，表示工序为路基开挖，级别为2，说明第2级，上级ID为2，说明安全责任单元履职清单主表中ID为2，名称为路基开挖为他的上一步工序
- 序号 (float)
- 字段1 (string)
- FGC_Creator (string)
- FGC_CreateDate (string)
- FGC_LastModifier (string)
- FGC_LastModifyDate (string)
- FGC_Rowversion (time)
- FGC_UpdateHelp (string)

表名: 带班作业记录表
字段:
- ID (int)
- 工程名称 (string): 带班工程就是工程名称，工程名称为类似的BM-00164编号
- 工单子表ID (int): 与每日管控计划_子表.ID对应，带班作业记录表.工单子表ID10就说明这个记录是每日管控计划子表ID10中的计划的带班记录
- 工单ID (int)
- 带班日期 (date)
- 带班开始时间 (string)
- 带班结束时间 (string)
- 带班作业工序及地点 (string): 地点在带班作业记录表中对应为此字段
- 带班期间工作内容 (string): 工作内容在带班作业记录表中对应为此字段，如果没有提及什么工作，那就是两个工作一起看。一起看时然后工作内容就是两个不同的内容
- 带班注意事项及要求 (string)
- 备注 (string)
- 带班人员 (string)：这个字段不是人名而是编号，所以要对应到大桥局人员信息表里面查真实人名
- 带班领导 (string): 这个字段不是人名而是编号，所以要对应到大桥局人员信息表里面查真实人名
- 是否联合 (string)
- 联合带班人员档号 (string)
- 电子签名 (string)
- 联合待办电子签名 (string)
- 状态 (string)
- 流程实例ID (string)
- FGC_Creator (string)
- 删除 (int)
- FGC_CreateDate (date): 时间在带班作业记录表中对应为此字段(类型为2025-02-23 14:37:40.402761)
- FGC_LastModifier (string)
- FGC_LastModifyDate (date)
- FGC_Rowversion (time)
- FGC_UpdateHelp (string)

表名: 每日管控计划
字段:
- ID (int)
- 计划日期 (date): 就是计划时间
- 管控单元id (int): 与责任单元字典.ID对应，因此每日管控计划.管控单元id可以了解管控计划的单元地方/部位，该单元负责人等等
- 所属项目 (string)
- 管控责任人 (string): 管控计划负责人档案编号
- 施工计划作业内容 (string): 管控计划内容在每日管控计划_子表.分项名称和每日管控计划.施工计划作业内容中间，例如管控计划内容为"掌子面正常开挖进尺，仰拱钢筋绑扎，二衬铺设防水板"可能是每日管控计划_子表.分项名称和每日管控计划.施工计划作业内容的一部分
- 状态 (string): 只有两种状态 已提交和暂缓，就是指计划是否提交了
- FGC_Creator (string)
- FGC_CreateDate (date)
- FGC_LastModifier (string)
- FGC_LastModifyDate (date)
- FGC_Rowversion (time)
- FGC_UpdateHelp (string)

表名: 每日管控计划_子表
字段:
- ID (int)
- 每日管控计划_ID (int): =每日管控计划.ID
- 分项名称 (string): 管控计划内容在每日管控计划_子表.分项名称和每日管控计划.施工计划作业内容中间，例如管控计划内容为"掌子面正常开挖进尺，仰拱钢筋绑扎，二衬铺设防水板"可能是每日管控计划_子表.分项名称和每日管控计划.施工计划作业内容的一部分
- 工序id (string)
- 工序id字符串 (string): 对应安全责任单元履职清单主表.ID，每日管控计划_子表.工序id字符串格式为105,131,132,133，可能工序id字符串有多个工序id
- 工序名称 (string)
- 管控人员 (string): 是指管控计划中的相关人员，风险研判，责任单元长，技术员，安全员，施工员机管员
- 风险研判 (string): 是指管控计划中风险
- 管控措施 (string): 是指管控计划中措施
- 关键工序 (int)
- 领导带班 (int): 为bool类型，1就是有，空就是无
- 跟班作业 (int)
- 关键工序档案 (string)
- 带班领导档案 (string): 就是带班人员领导的领导档案编号
- 跟班人档案 (string): 就是跟班人员领导的领导档案编号
- FGC_Creator (string)
- FGC_CreateDate (date)
- FGC_LastModifier (string)
- FGC_LastModifyDate (date)
- FGC_Rowversion (time)
- FGC_UpdateHelp (string)

表名: 跟班作业记录表
字段:
- ID (int)
- 工程名称 (string): 跟班工程也是工程名称，工程名称为类似的BM-00164编号
- 工单子表ID (int): 与每日管控计划_子表.ID对应，跟班作业记录表.工单子表ID10就说明这个记录是每日管控计划子表ID10中的计划的跟班记录
- 工单ID (int)
- 日期 (string)
- 班组 (string)
- 重点部位_关键工序_特殊时段情况 (string): 工作内容在跟班作业记录表中对应为此字段，如果没有提及什么工作，那就是两个工作一起看。一起看时然后工作内容就是两个不同的内容
- 安全质量__管控情况__ (string)
- 过程偏离或达不到安全质量目标情况及处理措施 (string)
- 跟班人员 (string)
- 电子签名 (string)
- 状态 (string)
- 跟班人员档案编号 (string)
- 流程实例ID (string)
- FGC_Creator (string)
- FGC_CreateDate (date)
- 照片 (string)
- 定位地址 (string): 地点在跟班作业记录表中对应为此字段
- 照片临时 (string)
- FGC_LastModifier (string)
- FGC_LastModifyDate (date)
- FGC_Rowversion (time): 时间在跟班作业记录表中对应为此字段(类型为2025-02-23 14:37:40.402761)，最近默认为从当前时间倒数一个月
- FGC_UpdateHelp (string)

关联关系与时间约定：
- 带班作业记录表.带班人员档案编号、带班领导档案编号，跟班作业记录表.跟班人员档案编号，每日管控计划_子表.带班领导档案、跟班人档案 = 大桥局人员信息表.档案编号
- 跟班作业记录表.跟班人员 = 大桥局人员信息表.姓名
- 每日管控计划_子表.每日管控计划_ID = 每日管控计划.ID
- 跟班作业记录表.工单子表ID，带班作业记录表.工单子表ID = 每日管控计划_子表.ID
- 每日管控计划_子表.工序id字符串 与 安全责任单元履职清单主表.ID 对应，并用逗号分隔
- 每日管控计划.管控单元id， 班前讲话班组字典.安全责任单元 = 责任单元字典.ID
- 班前讲话班组字典.班组 = 跟班作业记录表.ID
