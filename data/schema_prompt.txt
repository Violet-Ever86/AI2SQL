表名: 大桥局人员信息表
字段:
- ID (int)
- 档案编号 (string)
- 排序 (string)
- 所属部门 (string)
- 所属项目 (string)
- 姓名 (string)
- 岗位 (string)
- 职务 (string)
- 手机号 (string)
- 状态 (string)
- 正式员工 (int)
- 安全责任单元长 (int)
- 用户名 (string)
- 选中 (int)
- 履职经历 (string)
- 所属稽查队 (string)

表名: 安全责任单元履职清单主表
字段:
- ID (int): 对应每日管控计划_子表.工序id字符串
- 名称 (string): 对应工序名称，需要注意工序有顺序
- 级别 (int): 指顺序
- 上级ID (int): 对应安全责任单元履职清单主表.ID，指当前工序的上一级工序。
- 序号 (float)
- 字段1 (string)

表名: 带班作业记录表
字段:
- ID (int)
- 工程名称 (string): 带班工程等价于工程名称
- 工单子表ID (int): 对应每日管控计划_子表.ID。带班作业记录表.工单子表ID10代表该字段指每日管控计划子表.ID=10对应的带班记录
- 工单ID (int)
- 带班日期 (date)
- 带班开始时间 (string)
- 带班结束时间 (string)
- 带班作业工序及地点 (string)
- 带班期间工作内容 (string)
- 带班注意事项及要求 (string)
- 备注 (string)
- 带班人员档案编号 (string)
- 带班领导档案编号 (string)
- 是否联合 (string)
- 联合带班人员档号 (string)
- 电子签名 (string)
- 联合待办电子签名 (string)
- 状态 (string)
- 流程实例ID (string)
- 删除 (int)

表名: 每日管控计划
字段:
- ID (int)
- 计划日期 (date): 指计划时间
- 管控单元id (int): 对应责任单元字典.ID。每日管控计划.管控单元id可以了解管控计划的单元地点、部位、单元负责人
- 所属项目 (string)
- 管控责任人 (string): 管控计划负责人档案编号
- 施工计划作业内容 (string): 管控计划内容为每日管控计划_子表.分项名称或每日管控计划.施工计划作业内容
- 状态 (string): 只有两种状态:已提交和暂缓

表名: 每日管控计划_子表
字段:
- ID (int)
- 每日管控计划_ID (int): 等价于每日管控计划.ID
- 分项名称 (string): 管控计划内容为每日管控计划_子表.分项名称或每日管控计划.施工计划作业内容
- 工序id (string)
- 工序id字符串 (string): 对应安全责任单元履职清单主表.ID。每日管控计划_子表.工序id字符串格式为105,131,132,133
- 工序名称 (string)
- 管控人员 (string): 指管控计划中的相关人员，包括责任单元长、技术员、安全员、施工员、机管员
- 风险研判 (string)
- 管控措施 (string)
- 关键工序 (int)
- 领导带班 (int): 为bool类型
- 跟班作业 (int)
- 关键工序档案 (string)
- 带班领导档案 (string): 指带班人员领导的领导档案编号
- 跟班人档案 (string): 指跟班人员领导的领导档案编号

表名: 班前讲话班组字典
字段:
- ID (int)
- 排序 (int)
- 安全责任单元 (int): 与责任单元字典.ID对应
- 班组ID (string): 对应跟班作业记录表.班组
- 班组 (string): 指该班组的具体班组名称
- 班组长 (string): 指班组长名称
- 用户名 (string)
- 手机号 (string): 指班组长的手机号
- 有效性 (int)
- 班组成员 (string): 指班组成员
- 公司 (string): 指班组的公司
- 所属项目 (string): 指班组所属项目
- 是否正常作业 (int)
- 公司信用代码 (string)

表名: 责任单元字典
字段:
- ID (int)
- 自动编号 (string)
- 排序 (int)
- 所属项目 (string)
- 所属部门 (string)
- 责任单元名称 (string): 指班组负责的单元地点、部位
- 单元长姓名 (string): 指该单元负责人
- 档案编号 (string): 对应大桥局人员信息表.档案编号
- 单元长用户名 (string)
- 计划填报人员 (string)
- 工程类型 (string): 如路基工程,桥梁工程,隧道工程。为一个工程类型或多个工程类型混合
- 关联状态 (string)
- 关联时间 (string)
- 图标 (string)
- 工程类型名称 (string): 指该单元的类型
- 状态 (string): 指该单元施工或停工
- 用户 (string)
- 是_否删除 (int)

表名: 跟班作业记录表
字段:
- ID (int)
- 工程名称 (string): 跟班工程也是工程名称，工程名称为类似的BM-00164编号
- 工单子表ID (int): 对应每日管控计划_子表.ID，跟班作业记录表.工单子表ID10代表该记录指每日管控计划子表ID10中的计划的跟班记录
- 工单ID (int)
- 日期 (string)
- 班组 (string)
- 重点部位_关键工序_特殊时段情况 (string)
- 安全质量__管控情况__ (string)
- 过程偏离或达不到安全质量目标情况及处理措施 (string)
- 跟班人员 (string)
- 电子签名 (string)
- 状态 (string)
- 跟班人员档案编号 (string)
- 流程实例ID (string)
- 照片 (string)
- 定位地址 (string)
- 照片临时 (string)

关联关系与时间约定：
- 带班作业记录表.带班人员档案编号、带班领导档案编号，跟班作业记录表.跟班人员档案编号，每日管控计划_子表.带班领导档案、跟班人档案 = 大桥局人员信息表.档案编号
- 跟班作业记录表.跟班人员 = 大桥局人员信息表.姓名
- 每日管控计划_子表.每日管控计划_ID = 每日管控计划.ID
- 跟班作业记录表.工单子表ID，带班作业记录表.工单子表ID = 每日管控计划_子表.ID
- 每日管控计划_子表.工序id字符串 与 安全责任单元履职清单主表.ID 对应，并用逗号分隔
- 每日管控计划.管控单元id， 班前讲话班组字典.安全责任单元 = 责任单元字典.ID
- 班前讲话班组字典.班组 = 跟班作业记录表.ID
